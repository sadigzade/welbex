# Проект "Welbex"

## Стек технологий в проекте
+ React
+ Typescript

## Как запустить приложение?
Склонируйте проект к себе локально на компьютер: 
```
git clone https://github.com/sadigzade/welbex.git
```

Далее установите зависимости:
```
npm install
```

Запустите проект:
```
npm run start
```
